# **App Name**: Click Tracer

## Core Features:

- 3D Model Display: Display the 3D model using Three.js.
- Click Detection: Detect clicks on the 3D model.
- Coordinate Retrieval: Retrieve the x, y, z coordinates of the click location on the 3D model using a raycasting tool.
- Coordinate Display: Display the coordinates in a readable format, perhaps on a 2D overlay or beneath the 3D view.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey depth and precision.
- Background color: Very light blue (#E8EAF6), near-white, providing a clean backdrop to the 3D model.
- Accent color: Violet (#7E57C2) for highlighting the coordinate data and interactive elements.
- Body and headline font: 'Inter', a sans-serif font for clarity and legibility.
- Use simple, geometric icons to represent different functionalities. Consider wireframe icons in a style that aligns with 3D modeling.
- The 3D model should occupy the majority of the screen. Coordinate data should be clearly displayed in a fixed position, either below the model or as an overlay.
- Subtle highlighting on the 3D model when clicked, perhaps a brief color change or a pulsating effect.
